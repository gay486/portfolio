해외 웹 에이전시 소개 홈페이지

메인 페이지
 – 총 11 section으로 나눠지고 섹션별 다양한 기능을 넣어 제작
첫 번째 section은 마우스무브 기능과 각 이미지마다 애니메이션 효과를 구현하여 메인 슬라이드로 제작하였습니다.
네 번째 section은 내비게이션 버튼에 따라 갤러리 이미지의 배치가 다르도록 구현하였고, 이미지 클릭시 모달창이 열려 큰 이미지를 볼 수 있도록 제작하였으며,
마우스 오버시 숨겨진 텍스트가 보이고, 이미지 사이즈를 up 시키면서 filter : brightness 를 사용하여 이미지가 어두워지도록 구현하였습니다.
여섯 번째 section은 좌우 화살버튼, 인디게이터버튼, 터치스와이프로 반응형 슬라이드를 구현하였습니다.
그 외의 section들은 마우스 오버시 이미지 회전이나 size up 또는 position 과 opacity 효과를 구현하여 숨겨진 이미지나 텍스트가 보이도록 제작하였습니다.
또한, Parallax 효과를 구현하여 스크롤을 내리면 이미지와 텍스트가 나타나는 애니메이션을 구현하였습니다.

서브 페이지 1
 – 갤러리 형식으로 되어 있는 portfolio 서브 페이지 제작
내비게이션 버튼에 따라 갤러리 이미지의 배치가 다르도록 구현하였고, 이미지 클릭시 모달창이 열려 큰 이미지를 볼 수 있도록 제작하였으며,
마우스 오버시 filter : brightness 를 사용하여 이미지는 어두워지고, 아래 있던 텍스트는 이미지 위로 올라오도록 구현하였습니다.

서브 페이지 2
 – 회사 위치 및 소개하는 contact_us 서브 페이지 제작
이름,메일 등 정보를 입력하여 폼전송을 할 수 있도록 PHP 와 AJAX 그리고 정규표현식을 활용하여 제작하였습니다.
공통으로 스크롤을 내리면 스마일 아이콘을 사용한 goTop 버튼이 보여지고, 클릭시 페이지 상단으로 부드럽게 이동하도록 제작하였습니다.
