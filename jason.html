DETAIL

단 하나뿐인 나만의 럭셔리 여행,  "다른 여행"을 추구하는 국내 여행사 홈페이지

메인 페이지
 – 총 5 section으로 나눠지고 섹션별 다양한 슬라이드와 PHP 기반으로 사이트 제작
첫 번째 section은 전체를 fixed 로 설정하여 하단으로 스크롤시에도 위치가 항상 고정되도록 구현하였고,
인디게이터 버튼에 addClass 를 사용하여 각 슬라이드의 타이머 컨트롤에 맞춰 시각적으로 확인할 수 있는 움직이는 바도 구현하였습니다.
세 번째 section은 페이드 인아웃 반응형 슬라이드로 제작되었고, 인디게이터 버튼을 이미지로 만들어 클릭 시 해당 이미지가 크게 보이도록 구현하였습니다.
네 번째 section은 마우스 오버 효과와 인디게이터 버튼에 맞춰 이미지 1장씩 움직이도록 제작하였고, 반응형에 맞춰 슬라이드 개수가 변경되도록 구현하였습니다.

서브 페이지 1 , 2
 – AJAX를 구현하고 회원가입(Join), Login 서브 페이지 제작
회원가입 시 AJAX에게 넘겨 받은 입력된 폼 데이터를 PHP를 통해 서버에 전달하고 MYSQL에 생성한 해당 테이블에 저장하였으며,
MYSQL에 저장된 데이터베이스의 목록을 PHP로 가져와서 입력된 아이디와 비밀번호의 정보가 일치할 경우, AJAX를 통해 로그인이 되도록 제작하였습니다.

서브 페이지 3
 – 여행 테마를 소개하는 서브 페이지 제작
이미지를 나열하고 반응형에 맞춰 보여지는 이미지의 개수가 변경되도록 구현하였고, 마우스 오버시 이미지 테두리에 색상이 나타나도록 효과를 주었습니다.
